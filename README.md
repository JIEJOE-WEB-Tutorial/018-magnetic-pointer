## **Describe📋️**
在页面里面，有一个会跟随鼠标移动的选择框光标，当用鼠标hover按钮元素的时候，光标会变形、框选住整个按钮，并带有磁力吸附效果；在按钮范围内无论怎么移动鼠标、光标都会固定在按钮上

<br>

## **Notice⚠️**
- 该效果所参考的[原网站](https://www.xzadudu179.top/)
- 该效果用CSS的transition、对光标进行的位移以及变形的过渡，有能力的小伙伴可以使用GSAP或者其他动画库来代替transition、已提供更好的性能
